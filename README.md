Professional Portfolio | Besomba Boanerges
A modern, responsive multi-page personal portfolio website built to showcase web development skills, professional experience, and personal projects.
The goal of this project is to create a clean, modern, and responsive portfolio to present my work and connect with opportunities
 Live Demo
https://portfolio-rho-blond-c6u7pe2ki0.vercel.app/

Tech Stack
HTML: Semantic structure for accessibility and SEO.

CSS3: Custom styling with Flexbox, CSS Grid, and responsive Media Queries.

JavaScript : Interactive mobile navigation and dynamic UI elements.


 Key Features
Multi-Page Architecture: Separate dedicated pages for Home, About, Experience, Projects, and Contact.

Mobile-First Design: Fully responsive layout that adapts perfectly from desktop to smartphone screens.

Smooth Navigation: Interactive hamburger menu for mobile users and smooth-scroll transitions.

Working Contact Form: Integrated with client-side validation for seamless communication.

Professional Polish: Includes hover effects, glassmorphism header, and high-quality UI components.

 Project Structure
 index.html : Hero landing page
 about.html : Biography and personal details
 experience.html : Professional history 
 projects.html : Project images and live demos
 contact.html : Functional contact form
 style.css : Global styles and responsive design
── script.js         # Navigation logic and interactivity
